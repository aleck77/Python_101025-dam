# Контекст курса
Это домашние задания по базовому Python (циклы, функции, рекурсия, списки, словари)
и введению в анализ данных (numpy, pandas, простые визуализации).

# Как помогать по ДЗ
- Сначала помоги понять задание, только потом код.
- Не давай готовое полное решение сразу:
  - сначала наводящие вопросы;
  - затем подсказки по структуре функции;
  - затем пример решения, если пользователь явно попросил.

# Специфика Jupyter
- В .ipynb файлах:
  - ссылайся на ячейки как «ячейка с функцией <имя>».
  - если в ноутбуке есть несколько заданий, всегда переспроси, с каким работать,
    либо ориентируйся на текст запроса пользователя.

# Рекурсия
- Если задача про рекурсию (sum_digits, sum_nested и т.п.):
  - Объясни базовый и рекурсивный случаи.
  - Покажи дерево вызовов для 1–2 маленьких примеров.
  - Сравни с итеративным решением.
